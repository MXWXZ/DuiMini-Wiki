# UIContainer

## 容器

xml名：container  
基类：[UIControl](uicontrol.md)、IUIContainer

## 属性

| 属性 | 合法值 | 默认值 | 说明 |
| :---: | :---: | :---: | :---: |
| **transmouse** | [见表](uicontrol.md#transmouse-qu-zhi) | single | 是否穿透鼠标事件 |
| bgcolor | 颜色 | transparent | 背景颜色 |

## 公开函数

| 函数名 | 说明 |
| :---: | :---: |
| SetBGColor | 设置背景颜色 |
| GetBGColor | 获取背景颜色 |
