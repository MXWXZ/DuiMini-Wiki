# UIUtils通用类

本类为通用类，可以在程序中独立调用，详细内容参看源码注释。

| 函数名 | 作用 |
| :---: | :---: |
| GetCurrentDir | 取得当前目录（不包含末尾\） |
| GetTimeStr | 取得时间格式化字符串 |
| GetScreenSize | 取得屏幕大小 |
| GetWorkAreaSize | 取得工作区大小 |
| SafeTStr | 获取安全字符串 |
| SafeStr | 获取安全字符串 |
| SafeWStr | 获取安全字符串 |
| SafeBYTE | 获取安全BYTE |
| Convert2WStr | 字符串转换 |
| Convert2Str | 字符串转换 |
| Convert2TStr | 字符串转换 |
